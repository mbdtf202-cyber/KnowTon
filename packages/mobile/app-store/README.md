# App Store Submission Guide

This directory contains all assets and documentation required for submitting the KnowTon mobile app to Apple App Store and Google Play Store.

## Submission Checklist

### Pre-Submission
- [ ] All features tested on physical devices
- [ ] App icons prepared (all sizes)
- [ ] Screenshots captured (all required sizes)
- [ ] App descriptions written (all languages)
- [ ] Privacy policy published
- [ ] Terms of service published
- [ ] App Store Connect account ready
- [ ] Google Play Console account ready

### iOS Submission
- [ ] App ID created in Apple Developer Portal
- [ ] Provisioning profiles configured
- [ ] App Store Connect app created
- [ ] Build uploaded via Xcode or Transporter
- [ ] App metadata completed
- [ ] Screenshots uploaded
- [ ] App submitted for review

### Android Submission
- [ ] Signing key generated
- [ ] App bundle (AAB) built
- [ ] Google Play Console app created
- [ ] Store listing completed
- [ ] Screenshots uploaded
- [ ] Content rating completed
- [ ] App submitted for review
